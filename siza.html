Siza Challenge


Next 4 challenges are about seasons. This word comes from French "saison" and is similar to Romanian "sezon". Maj word for season represents a noun and is read: /siza/. Also you can refer to a season using the more common word "tepa" = time. Dictionary:
  • foro-tepa: flourished time = spring
  • soro-tepa: sunny time = summer
  • luvo-tepa: rainy time = autumn
  • nejo-tepa: snowy time = winter

Challenge 1:

Translate:
  • The trees were blooming.
  • It was spring; my favorite season.
Dictionary:
  • Arba = tree
  • sh floru = to bloom
  • fevo = favorite
Maj:
  • Le Arbe sonu-pa floro.
  • Lo sonu-pa foro-tepa; mi fevo siza.

Challenge 2:

Translate:
  • Outside it is now suny and all fields are green.
  • It is the summer. I'm going on vacation.
Dictionary:
  • eksa = outside
  • felda = field
  • vatsa = vacation
  • andu = going
  • soro = sunny
  • oma felde = all fields
  • sonu geno = are green
Maj:
  • eksa oi sonu soro ci oma felde sonu geno;
  • lo sonu la soro-tepa; ea andu yn vatsa;

Challenge 3:

Translate:
  • The leaves of all trees soon will be yellow.
  • Without any doubt it will be the autumn.
Dictionary:
  • le xoje = the leafs
  • ome Arbe = all trees
  • sonu-va = will be
  • ielo = yellow
  • seo = soon
  • uko = without
  • duda = doubt
  • oce = any
Maj:
  • le xoje de ome Arbe seo sonu-va ielo;
  • uko oce duda lo sonu-va luvo-tepa;
Note: In this example we have used the adverb "seo" = soon before the verb sonu-va. The adverb can be placed in front of the verb. Another issue is that ielo is an adjective placed after the verb sonu-va. In this case the adjective is referring to the subject xoje.

Challenge 4:

Translate:
  • In the winter there will be a lot of snow.
  • We must wear warm clothes and gloves.
Dictionary
  • fil = much
  • neja = snow
  • vetu= to wear/ to dress
  • cado = warm
  • ropa = clothes
  • gune = gloves
Maj:
  • in la nejo-tepa sonu-va fil neja;
  • noi dovu sh vetu rope cado cy gune;

Note: Observe the order of words in Maj is OA: (Object Adjective): "ropa cado". You can use "cado-ropa". This is a composite word created using the A-O pattern. This is the secret of inversion, you must use dash between the two words

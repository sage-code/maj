What is Maj?

Maj is an artificial auxiliary language. It was designed to improve our ability to create and implement challanging projects. This particular project require multiple artistic, engineering and programming skills. We have chosen to make an artifficial language becouse it is difficult. We are continuously learning new things by making Maj. You are invited to join us.

Maj Grammar

Next we will describe Maj language in details using examples. Maj should be more easy to learn compared to other languages but it will require many hours of study. This language is not yet spoken by anybody so you do not have someone to help you yet. Next document was created to introduce Maj to other conlang creators last year (2020).

Maj Dictionary

We use Google docs to create a dictionary. These include translation in 6 languages. In the future we will create a database and a translation form. For now these spreedsheets is all we have. So far Maj has about 3700 words and slowly we add new words.

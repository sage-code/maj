Maj Adverbs

Maj uses free order grammar. Default order is SVA: (Subject Verb Adverb), but it can be inverted: VSA (Verb Subject Adverb) to create the imperative and interrogative cases.

Adverb comparison:

Adverbs can change coda from "o" into "h" to emphasize a quality or property of one object over another. We also use the preposition "di" = "than" to make a comparison between subject and object.
  • Ea lofu cnelh du toa. = I run faster than you.
  • Ta lofu lenth du moa. = You run slower than me.

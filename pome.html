    • Challenge 1:

    Translate:
    • How many bananas are in a bunch?
    • One stem of bananas can have 3 up to 20 rows.
    • Each row of bananas is called a “hand” and is made up of 14 to 20 "fingers".
    Dictionary
    • vi fil = how many
    • buke = bunch
    • mana = hand
    • dege = fingers
    • ryna= row
    • stega = stem
    • posu = can
    • xaru = have
    • voku = call
    • ogi = each
    • faru de = made up of
    Maj:
    • vi file Bane sonu yn una buka?

    • ogi ryna de Bane se voku una "mana" ci sono faru de dive zu bedi dege;
